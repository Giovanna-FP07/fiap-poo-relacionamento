# Sistema de Bilhete Único - Atividade de Programação Orientada a Objetos

## Descrição do Projeto

Este projeto simula o funcionamento de um Bilhete Único, representando regras básicas de validação de entrada em transporte público.

Esta é a junção da atividade 1, 2 3, 4 e 5 desenvolvida nas últimas aulas. O sistema foi desenvolvido utilizando Programação Orientada a Objetos (POO) em Java, aplicando conceitos como implementação de classes, métodos. getters/setters, encapsulamento, construtores e relacionamento.

---

## Como Executar o Projeto

1. Clone este repositório

```
 https://github.com/Giovanna-FP07/fiap-poo-relacionamento.git
```

2. Abra o projeto em sua IDE de preferência

3. Execute o arquivo: "SistemaPrincipal.java"

4. Execute o arquivo: "TesteBilheteUnico.java"
